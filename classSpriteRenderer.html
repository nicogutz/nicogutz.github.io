Detailed Description

The SpriteRenderer class handles the conversion of data to GamePixmapItems. It also animates behaviors not generalizable such as death, which in this view change the frame from the sprite.

GamePixmapItem * SpriteRenderer::renderGameObject ( QMap< DataRole, QVariant >  data)
overridevirtual

renderGameObject Overloaded from Renderer. Makes a new GamePixmapItem and adds the appropriate sprite. This function is only run once in the SpriteRenderer. Which means that changes to the objects are applied to the same sprite. The other renderers do not have this feature since the bulk of the changes are very difficult to animate using parameters and not generalizable.

Returns
