class  Movement
 The Movement class handles the actions of moving. It is an abstract class. All the functions return false by default so that objects that can move cannot be walked through by default, and objects that can be stepped on cannot be moved. Also, some objects can be stepped on by enemies but not trigger their behaviors. E.G. Doorways can be stepped on by enemies (so they return true on isSteppable and getSteppedOn) but do not start a new level. More...
